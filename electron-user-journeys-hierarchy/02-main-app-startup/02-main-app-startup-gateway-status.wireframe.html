User outcome, constraints, evidence gaps
User outcome

Startup feels dependable while health checks are still running. Users can see recent sessions and continue available work.

Constraints

More visible health states require careful copy. Startup should not show competing prompts.

Evidence gaps

Verify which actions remain available offline or degraded, and how disabled actions explain why.
