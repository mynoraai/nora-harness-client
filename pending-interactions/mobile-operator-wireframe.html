Journeys Wireframes References Dependencies

Journey Summary

1

Pair phone

Scan or paste setup code, wait for Gateway approval, then open the session list.

2

Choose session

Open session list, mark desktop focus, and switch the mobile selected session.

3

Observe or stop

Running sessions show stream and tool cards. Stop is the only run action.

4

Send when idle

After terminal state and history refresh, the composer returns for normal send.
