# L3 Computer-Use Scenarios

Source rows: `ONB-*`, `MAIN-*`, `CHAT-*`, `CODE-*`, `AUTH-*`, `SET-*`, `BND-*`

Status: Draft target matrix

There is no wired Electron e2e harness yet. These scenarios are written so either Playwright/Electron or a computer-use driver can implement them once stable selectors exist.

## Harness Rule

Use role, accessible name, `aria-label`, or `data-testid`. If a target lacks a stable selector, mark the scenario as blocked by `No stable selector`; do not rely on pixel coordinates.
