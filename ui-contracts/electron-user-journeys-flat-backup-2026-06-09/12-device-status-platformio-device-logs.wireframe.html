Journey notes

User outcome, constraints, evidence gaps
User outcome

Hardware readiness is visible before upload or preview work. Logs become part of the workspace instead of a hidden terminal-only task.

Constraints

Hardware status can be noisy when devices flap. Install guidance may differ by operating system.

Evidence gaps

Should the Device tab be shown for every workspace or only hardware-classified workspaces?
