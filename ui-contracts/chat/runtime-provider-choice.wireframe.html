3. Provider 不是 runtime

Provider 是账号/API/model 来源；runtime 是谁执行 coding session。用户需要能理解这两层。

Runtime layer

  • NoraHarness Default / OpenClaw
  • Claude Code
  • Codex

Provider/model layer

  • Anthropic, OpenAI, Google
  • OpenRouter, MiniMax, Local
  • Enabled models in Settings

可见文案对比

Expose OpenClaw

  • Pros: exact match to engineering registry.
  • Pros: easier support/debug mapping.
  • Risk: exposes inherited backend as product brand.

NoraHarness Default

  • Pros: keeps NoraHarness as product center.
  • Pros: avoids implying OpenClaw is another vendor app.
  • Risk: needs secondary support copy for logs/debug.
