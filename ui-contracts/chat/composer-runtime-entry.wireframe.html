PM decision

Runtime 选择器 detail

Today

  • 按钮显示 runtime label。
  • 弹窗列出 Runtime group。
  • 锁定时 title 显示 Runtime is locked for this session。

Proposed

  • 在 row 右侧显示 Ready / Auth / Install。
  • 锁定时展示短原因，而不只靠 tooltip。
  • 保留一个入口，不另开 runtime panel。
PM decision

模型选择 detail

Today

  • runtime 变化会重新拉模型目录。
  • 保存 session 可 patch model。
  • 无模型时按钮显示 No enabled models。

Proposed

  • 把 runtime model 和全局 default model 的关系放进错误恢复文案。
  • Unavailable 模型给用户一个明确动作：换模型或去 Settings。
PM decision

附件入口 detail

Today

OpenClaw 不限制 accept；ACP runtime 只接受 JPEG、PNG、GIF、WebP、HEIC、HEIF。 代码里当前 runtime 都支持附件，但 runtime 能力仍是产品可见边界。

Proposed

PM 决定附件按钮是“永远显示并在 runtime 不支持时解释”，还是“按 runtime 隐藏/禁用”。

Proposed language

状态语言

用户可见

  • Ready to send
  • Starting runtime
  • Streaming
  • Runtime locked
  • Model unavailable

工程状态

  • idle / starting / ready / running
  • binary-missing / auth-required
  • handshake / permission / workspace / gateway
